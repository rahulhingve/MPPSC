✅ Advantages
  • Cheapest — lowest cost cable
  • Easy to install
  • Supports both Analog & Digital signals
  • Fault tolerant — one fault doesn't break whole network
❌ Disadvantages
  • Not for long distance communication
  • Low data transmission rate
  • Speed decreases as connections increase

  • More common, cheaper
STP — Shielded Twisted Pair
  • Has extra metal/foil shield
  • Better protection from interference
🎯 Exam Tip
Q: "If fault occurs in network, entire network fails" → This statement is FALSE for Twisted Pair. Entire network does NOT fail.

04 Wireless Communication — Long Range (5 Types) Important ›

All wireless communication uses Electromagnetic Waves (विद्युत चुंबकीय तरंगें). EM Spectrum: Radio → Microwave → Infrared → Visible → UV → X-ray → Gamma.

#TypeHindi NameKey Feature
1Space Wavesअंतरिक्ष तरंगेंTravel in straight line (line of sight), used in TV broadcasting
2Sky Wavesआकाशीय तरंगेंReflect off ionosphere, used in AM radio (long distance)
3Ground Wavesभूमि तरंगेंTravel along Earth's surface, used in AM radio (short distance)
4Satellite Communicationउपग्रह संचारGeostationary satellites, GPS, DTH television, very long range
5Microwaveमाइक्रो तरंगेंHigh frequency, line of sight, used in mobile networks & radar
📌 Remember
Long Range = Satellites, Sky waves, Ground waves, Space waves, Microwave. All use EM spectrum.

06 By Capacity — Baseband, Voiceband, Broadband Important ›
📌 Baseband

Only one signal at a time. Low bandwidth. Uses digital signal. Used in LAN (Ethernet). All bandwidth dedicated to single channel.

📞 Voiceband

Designed for voice transmission. Low bandwidth (300 Hz – 3400 Hz). Traditional telephone networks. Used in modem connections.

🚀 Broadband

Multiple signals simultaneously. High bandwidth. Used in internet, cable TV, DSL. Supports voice, data, video at same time.

08 By Permission — Internet, Intranet, Extranet Important ›
🌍 Internet
  • Public network — anyone can access
  • No single owner
  • Global network of networks
  • Uses TCP/IP protocol
  • Example: Google, YouTube
🏢 Intranet
  • Private network — only employees
  • Owned by an organization
  • Internal communication & data
  • Secured with passwords/firewall

  • Company shares info with trusted partners/suppliers/customers

09 Other Types — Serial, Parallel, Unicast, Multicast, Broadcast, Analog, Digital Important ›

Serial vs Parallel Communication

📤 Serial Communication
  • Data sent one bit at a time
  • Uses single wire
  • Slower but reliable for long distance
  • Example: USB, RS-232
📤📤 Parallel Communication
  • Data sent multiple bits simultaneously
  • Uses multiple wires
  • Faster but limited to short distances
  • Example: Old printer port

Unicast vs Multicast vs Broadcast

TypeSenderReceiverExample
Unicast1 device1 specific deviceWhatsApp message to one person, Email
Multicast1 deviceSelected groupWhatsApp group message, Video conference
Broadcast1 deviceALL devices in networkTV broadcast, Radio, ARP packet in network

Analog vs Digital Signal

〜 Analog Signal
  • Continuous wave — infinite values
  • Represented by sine wave
  • More susceptible to noise
  • Examples: Human voice, AM/FM radio
  • Old telephone system
▐▐ Digital Signal
  • Discrete values — only 0 or 1
  • Represented as square wave
  • More reliable, less noise
  • Examples: Computer data, WiFi
  • Modern communication systems

11 Data Transmission Modes — Simplex, Half Duplex, Full Duplex Most Important ›
Simplex
A ──→ B
One-way communication only. Data flows in ONE direction only. Receiver cannot send back.
📺 TV, Radio, Keyboard → Computer
Half Duplex
A ⇄ B
Two-way communication but NOT simultaneously. Only one can send at a time. Must take turns.
📟 Walkie-Talkie, Fax Machine
Full Duplex
A ⇔ B
Two-way communication simultaneously. Both can send AND receive at same time.
📱 Mobile Phone, Video Call, Telephone
ModeDirectionSimultaneous?Example
SimplexOne-way only❌ NoTV, Radio, Keyboard, Mouse
Half DuplexBoth ways (alternate)❌ No (takes turns)Walkie-Talkie, CB Radio, Fax
Full DuplexBoth ways✅ YesMobile call, Telephone, Video call
🎯 Most Asked
"Walkie-talkie uses which mode?" → Half Duplex | "Telephone uses?" → Full Duplex | "TV remote uses?" → Simplex
