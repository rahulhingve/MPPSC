💻 Software - System vs Application

Software = instructions/programs ka set, jinke basis par computer kaam karta hai. Class mein Ada Lovelace ko first programmer ke roop mein yaad karaya gaya.

System Software

  • Electronic device ke fundamental/base software.
  • Computer ko on/off, run, configure aur maintain karne mein help.
  • User aur hardware ke beech interface banata hai.
  • Examples: Operating System, Utility Software, Language Translator, Firmware, Drivers.

Application Software

  • Specific user work ke liye banaye jaate hain.
  • Horizontal/General purpose: common tasks ke liye.
  • Vertical/Special purpose: kisi special domain/task ke liye.
  • Examples: PowerPoint, Photoshop, editor, games, browser apps.

🌐 Internet Basics

Internet = International Network; network of networks; duniya bhar ke computer networks ko connect karta hai.

Computer A→ TCP breaks packets Internet→ IP finds address Computer B

Definition

Duniya bhar ke computer networks ko common protocol ke through connect karna Internet hai. Isliye ye Wide Area Network ka example hai.

Working

TCP message ko packets mein todta hai; IP har connected device ko unique address deta hai; ISP internet service provide karta hai.

Web TypeMeaningImportant Fact
Surface WebPublicly accessible information; no permission needed.Class mein approx 4% bataya; examples Google, Wikipedia, Facebook, YouTube.
Deep WebLogin/permission required data.Companies, government, paid subscriptions; biggest part.
Dark WebNormal browser se access nahi; TOR required.Illegal activities, hackers, crypto payments mentioned.
🔥 Exam Hot
  • Father of Internet: Vint Cerf, because TCP/IP protocol development se associated.
  • Internet ka koi single owner nahi; regulatory organizations standards/protocols manage karte hain.
  • Telnet login ke liye; Usenet filtering/indexing; Pushnet electronic bulletin board message ke liye class mein bataya gaya.

📡 Internet Devices

Internet access ke liye ISP, server, modem/router aur device ki need hoti hai.

Modem

MO + DEM = Modulation + Demodulation. Telephone/analog signal aur computer/digital signal ke beech conversion karta hai.

  • Digital → Analog = Modulation
  • Analog → Digital = Demodulation

Router

Internet ko wired ya wireless way mein aage distribute karta hai; home Wi-Fi routers mein modem component bhi hota hai.

Repeater

Long cables/transmission mein signal weak na ho, isliye repeat/strengthen karta hai.

Hub

Multiple devices ko ek local network mein connect karne ka simple device.

Multiplexer

Low-speed transmission lines ko high-speed line se connect karta hai.

Gateway / Bridge

Different networks/protocols ko connect karne mein help. Gateway protocol converter ki tarah kaam karta hai.

🚀 Net Neutrality, Space Internet & Proxy Server

Lecture ke last current/other facts: net neutrality, Starlink space internet aur proxy server.

Net Neutrality

Internet service provider sabhi users/services ke saath equal behavior kare, kisi app/site ko unfair speed/blocking advantage na de. Isse right to internet type concept se link kiya gaya.

Space Internet / Starlink

Satellite ke through internet provide karna. SpaceX/Elon Musk ka Starlink project; low earth orbit satellites; remote areas tak internet. Concern: space debris.

Proxy Server

Client aur server ke beech intermediary. User ki request pehle proxy server ko jaati hai, proxy doosre IP se server se data laata hai.

Proxy Server BenefitExplanation
IP hideUser ka original IP address directly server tak nahi jaata.
Blocked sites accessClass mein blocked website access possible bataya.
Speed improveRepeated info proxy cache memory mein store ho sakti hai.
MonitoringNetwork use monitor karne mein help.
SecurityPrivate network par attack risk reduce karne mein help.
