🌳 Number Family Tree — Dada-Beta Analogy
संख्या परिवार — पूरा वंशवृक्ष

👴 Dada = Complex Number (सम्मिश्र संख्या)

Sabse upar Complex Number hai — ye poore number family ka dada hai.

Dada ke 2 bete hain:

✅ Beta 1: Real Number (वास्तविक)

Shareef beta — jo number line pe represent ho sake.

🎯 Isko padhna hai! Exam mein isi se questions aate hain.

❌ Beta 2: Imaginary (काल्पनिक)

Kalpana wala beta — number line pe represent nahi ho sakta.

❌ Isko padhna nahi hai!

Complex (सम्मिश्र) — DADA 👴
├── Real (वास्तविक) — Shareef Beta ✅
│   ├── Rational (परिमेय)
│   │   ├── Integer (पूर्णांक) → ...-2,-1,0,1,2...
│   │   │   ├── Whole (पूर्ण संख्या) → 0,1,2,3...
│   │   │   │   └── Natural (प्राकृतिक) → 1,2,3...
│   │   ├── Fraction (भिन्न) → 1/2, 3/4
│   │   └── Decimal → 0.5, 0.333...
│   └── Irrational (अपरिमेय) → √2, √3, π
└── Imaginary (काल्पनिक) — Kalpana Beta ❌
Aditya Sir ka rule: Jaise Ambani ke 2 bete hain — Mukesh aur Anil. Focus Mukesh pe hai, Anil ko koi nahi puchta. Waise hi Real pe focus karo, Imaginary ko chhodo!
🔍 Real vs Imaginary Numbers
वास्तविक बनाम काल्पनिक संख्या

✅ Real Number (वास्तविक संख्या)

Definition: Those numbers which can be represented on number line

जिनको संख्या रेखा (number line) पर दर्शाया जा सके = वास्तविक संख्या

Jaise: 2, -3, 0.5, √2, π — ye sab number line pe aa sakte hain.

❌ Imaginary Number (काल्पनिक संख्या)

Definition: Jab root ke andar minus aaye = Imaginary

√(-1) = i   √(-4) = 2i

Jaise tum kisi ko kitna pyaar karte ho — iska koi measuring scale nahi hai. Waise hi ye sirf kalpana mein hai!

🔗 Complex = Real + Imaginary

a + ib → jahan a = Real part, ib = Imaginary part

Dono ko mila do = Complex Number ban jaata hai.

Exam ke liye: Imaginary/Complex ke baare mein bilkul nahi padhna! Sirf Real Number padhna hai. But "Complex/Imaginary kya hai" ye samajhna zaroori hai taaki question aaye to confuse na ho.
⚖️ Rational vs Irrational — Deep Clarity
परिमेय बनाम अपरिमेय — गहरी समझ — MOST IMPORTANT

Real Number ke 2 bete: Rational aur Irrational

✅ Rational Number (परिमेय संख्या)

Origin: "Rational" shabd aaya hai "Ratio" se — matlab jisko ratio/fraction mein likh sako.

Definition: Jo p/q ke form mein ho, jahan p, q = Integer aur q ≠ 0

Rational ke 2 properties (Pizza Example 🍕):

TypeHindiKya hota hai?Example
Terminating शांत दशमलव Decimal ruk jaata hai — iska ek ant hai 1/2 = 0.5, 1/4 = 0.25
Non-Terminating Repeating अशांत आवर्ती दशमलव Rukta nahi but pattern repeat hota hai 1/3 = 0.333..., 1/6 = 0.1666...
Denominator Rule: Agar neeche wale number mein sirf 2 ya 5 ke multiples hain to Terminating hoga. Baaki mein Non-Terminating Repeating.

❌ Irrational Number (अपरिमेय संख्या)

Definition: Jisko p/q mein likh hi nahi sakte.

Property: Non-Terminating + Non-Repeating

ना रुकता है, ना दोहराता है — बस चलता जाता है!

ExampleValueKya ho raha?
√21.41421356...Na ruka, na repeat hua
√31.73205080...Na ruka, na repeat hua
π3.14159265...Na ruka, na repeat hua

⚡ Ek Nazar Mein:

CategoryDecimal BehaviourExampleType
Ruk jaayeTerminating0.5, 0.25Rational ✅
Na ruke but repeat kareNon-Terminating Repeating0.333..., 0.1212...Rational ✅
Na ruke, na repeat kareNon-Terminating Non-Repeating√2, √3, πIrrational ❌
Misconception: Bahut log sochte hain "jisme root laga = Irrational". Ye GALAT hai! √4 = 2 (Rational hai). Sirf wo root Irrational hai jiska answer integer mein na aaye!
🪤 Trap 1: Root Wale Confusing Cases
जड़ (Root) वाले भ्रामक प्रश्न — EXAMINER'S FAVOURITE

Duniya kya yaad rakhti hai? → "Jisme root laga = Irrational". Ye GALAT hai!

📏 Golden Rule: Root solve ho = Rational, Root na solve ho = Irrational

ExpressionSolveRoot hata?Type
√31.732...❌ NahiIrrational
√4= 2 = 2/1✅ HaanRational
√9= 3 = 3/1✅ HaanRational

💣 Decimal Root Trap — √0.4 vs √0.04 vs √0.009

Aditya Sir ka Master Rule:

Root wale decimal mein 2 Conditions check karo:

1️⃣ Decimal ke baad Even places (2,4,6,8...) honi chahiye

2️⃣ Us number ka perfect square root aa jaana chahiye

Dono conditions puri ho = Rational | Koi ek bhi fail = Irrational

🔍 Examples — Step by Step:

RootDecimal ke baad digitsEven places?Root solve?Type
√0.4 1 digit (just "4") ❌ Odd (1) — Irrational
√0.04 2 digits ("04") ✅ Even (2) ✅ √4=2 → 0.2 Rational ✅
√0.009 3 digits ("009") ❌ Odd (3) — Irrational
√0.0009 4 digits ("0009") ✅ Even (4) ✅ √9=3 → 0.03 Rational ✅
√0.07 2 digits ("07") ✅ Even (2) ❌ √7 solve nahi hota Irrational
√0.09 2 digits ("09") ✅ Even (2) ✅ √9=3 → 0.3 Rational ✅
Quick Yaad Trick:
• Decimal ke baad Odd places → ALWAYS Irrational (even check karne ki zarurat nahi)
• Decimal ke baad Even places → Tab bhi check karo ki us number ka root banta hai ya nahi
• Jaise √0.07 → Even places hain BUT 7 ka root nahi banta → Irrational
🪤 Trap 2: Operations Trick (√75/√3)
संक्रिया वाला जाल — दो अपरिमेय = परिमेय? — TRICKY

Duniya sochti hai: Irrational ÷ Irrational = Irrational. Hamesha nahi!

🔍 Step by Step: √75 ÷ √3 = ?

Step 1: √75 → Kya 75 ka root hota hai? → Nahi → Irrational

Step 2: √3 → Kya 3 ka root hota hai? → Nahi → Irrational

Duniya sochti hai: Dono Irrational → Answer bhi Irrational. GALAT!

Step 3: Simplify karo → √75 = √(25 × 3) = √25 × √3 = 5√3

Step 4: Divide → 5√3 ÷ √3 = 5

Step 5: 5 = 5/1 → p/q form → RATIONAL! ✅

Aditya Sir: "Agar main examiner hota aur tumhe fansata — to dono numbers Irrational deta aur bolta batao kya hai? Tum guarantee kehte Irrational aur wahi fans jaate!"

Lesson: Pehle simplify/solve karo, phir decide karo ki Rational hai ya Irrational!
🪤 Trap 3: π (Pi) vs 22/7 Confusion
पाई (π) और 22/7 का भ्रम — EXAM TRAP

π (Pi Symbol)

π = 3.14159265...

Non-Terminating + Non-Repeating

= IRRATIONAL ❌

22/7 (Fraction)

22/7 = 3.142857142857...

Non-Terminating BUT Repeating (142857 repeat)

= RATIONAL ✅

Examiner ka trap: Question mein "22/7" likhke puchta hai "ye Rational hai ya Irrational?" — Agar tum sochte ho 22/7 = π = Irrational... to GALAT!

22/7 sirf π ki approximate value hai, π nahi hai!
• 22/7 → p/q form mein hai → Rational ✅
• π → p/q mein likh hi nahi sakte → Irrational ❌

📌 Integer (पूर्णांक)

Jiske neeche mein kuch bhi na ho — no fraction, no decimal.

..., −3, −2, −1, 0, +1, +2, +3, ...

Number line pe aasani se represent ho sake = Integer

📏 Number Line Visualization:

←────3──(−2)──(−1)──[0]──(+1)──(+2)──(+3)───→

  ◄── Negative (ऋणात्मक)   ↕   Positive (धनात्मक) ──►
                             Zero = NEUTRAL

⭐ Zero (शून्य) Special Case:

Zero na positive hai, na negative — Zero is NEUTRAL!

शून्य — ना बायें आता है, ना दायें. बीच में बैठा है.

Aditya Sir: "Negative ki paribhasha — jo zero ke left mein aaye. Positive — jo right mein aaye. Zero na left mein hai na right mein. Center mein hai!"

WHOLE NUMBER (पूर्ण संख्या)

Zero + sab Positive

0, 1, 2, 3, 4, 5...

Zero ke saath count = Whole

NATURAL NUMBER (प्राकृतिक संख्या)

Sirf Positive (no zero)

1, 2, 3, 4, 5...

Zero ke bina count = Natural

Aditya Sir ka formula:
• 0 ke saath ginti karo = Whole Number (पूर्ण संख्या)
• 0 ke bina ginti karo = Natural Number (प्राकृतिक संख्या)
Bas itna yaad rakho — life mein kabhi nahi bhuloge!

📝 PYQ Practice — CPO/CGL/CHSL 2025
Previous Year Questions with Explanations
📌 CPO 2025 — QUESTION 1
Which of these number set is NOT a subset of Real Numbers?
इनमें से कौन सा संख्या समूह वास्तविक संख्याओं का उपसमूह नहीं है?
  • A) Irrational Numbers
  • B) Rational Numbers
  • C) Complex Numbers
  • D) Integers

✅ Answer: C) Complex Numbers

Logic: Family tree yaad karo:

Complex = DADA hai. Real uska BETA hai.

Question pucha: Real ka subset (beta) kaun nahi hai?

• Irrational → Real ka beta ✅

• Rational → Real ka beta ✅

• Integer → Rational ka beta (matlab Real ka pota) ✅

• Complex → Real ka BAAP hai, beta nahi! ❌

Baap kabhi bete ka subset nahi hota! 💥

📌 CPO 2025 — QUESTION 2
The set of real numbers includes which of the following?
वास्तविक संख्याओं के समूह में कौन शामिल है?
  • A) Only Rational Numbers
  • B) Both Rational and Irrational Numbers
  • C) Only Irrational Numbers
  • D) Only Integers

✅ Answer: B) Both Rational and Irrational

Real ke 2 bete hain — Rational AUR Irrational. Dono isme shamil hain!

📌 CPO 2025 — QUESTION 3
Square root of any prime number is:
किसी भी अभाज्य संख्या का वर्गमूल होता है:
  • A) Rational
  • B) Irrational
  • C) Integer
  • D) Complex

✅ Answer: B) Irrational

Prime numbers: 2, 3, 5, 7, 11...

√2 = 1.414... | √3 = 1.732... | √5 = 2.236... | √7 = 2.645...

Kisi bhi prime number ka root kabhi solve nahi hota → Always Irrational!

📌 CGL 2025 — QUESTION 4
Which number is real but NOT rational?
कौन सी संख्या वास्तविक तो है लेकिन परिमेय नहीं है?
  • A) 3/4
  • B) −7
  • C) √7
  • D) 0.5

✅ Answer: C) √7

Real BUT NOT Rational = matlab Irrational puch raha hai!

• 3/4 → p/q form → Rational ✅

• −7 → Integer → Rational ✅

• √7 → Root solve nahi hota → Irrational! ❌

• 0.5 → Terminating decimal → Rational ✅

📌 CGL 2025 — QUESTION 5
Which number is neither rational nor integer?
कौन सी संख्या ना तो परिमेय है ना ही पूर्णांक?
  • A) 0
  • B) −5
  • C) √2
  • D) 7

✅ Answer: C) √2

Jo Rational hi nahi hai wo Integer kaise hoga? Integer to Rational ka beta hai!

√2 = Irrational → Not Rational → Not Integer → Neither!

📌 SSC 2025 — QUESTION 6 (Operations)
The sum of a rational number and an irrational number is:
एक परिमेय और एक अपरिमेय संख्या का योग होता है:
  • A) Always Rational
  • B) Always Irrational
  • C) Always Integer
  • D) Sometimes Rational, Sometimes Irrational

✅ Answer: B) Always Irrational

R + IR = ALWAYS Irrational (no exception!)

Example: 2 + √3 = 3.732... → Non-terminating Non-repeating → Irrational

📌 SSC 2025 — QUESTION 7 (Tricky!)
The product of a rational number with an irrational number is:
एक परिमेय को अपरिमेय से गुणा करने पर:
  • A) Always Rational
  • B) Always Irrational
  • C) Always Integer
  • D) Sometimes Rational, Sometimes Irrational

✅ Answer: D) Sometimes Rational, Sometimes Irrational

Operations Table yaad karo!

• 2 × √3 = 2√3 → Irrational ❌

• 0 × √3 = 0 → Rational! ✅

Jab R = 0 ho, result Rational aa jaata hai. Isliye "Sometimes" wala answer sahi hai.

💡 Jo log "Always Irrational" select karte hain — unke number katte hain! Ye Aditya Sir wala exception hai.

📌 SSC 2025 — QUESTION 8
If x is irrational and y is irrational, then x + y is:
यदि x अपरिमेय है और y अपरिमेय है, तो x + y होगा:
  • A) Always Irrational
  • B) Always Rational
  • C) Irrational unless x = −y, then Rational
  • D) Always Integer

✅ Answer: C)

Usually IR + IR = Irrational

BUT: Agar x = −y → x + y = 0 → Rational!

Example: √2 + (−√2) = 0 ← ye Rational hai!
